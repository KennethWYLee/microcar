Real Study Scene

實際上課時，大概就是這樣

先看一下真實學習畫面，你會更知道自己現在在做什麼，也比較不會被陌生的小車和電腦環境嚇到。

學生會一邊看電腦上的程式，一邊看桌上的小車反應，所以先認得 `Run`、`Shell` 和按鍵就很重要。

整堂課不是考你背很多內容，而是讓你一步一步完成連線、輸入指令、讓車子動起來這三件事。

你現在要記住的事

  • 每個按鍵都有一個動作。
  • 每個動作後面，通常都會對應一個函式(Function)。

鍵盤控制(Keyboard Control)

按下按鍵，小車就做動作

cmd = input("Command: ").strip().lower()

if cmd == "w":
    forward()
elif cmd == "s":
    backward()
elif cmd == "a":
    turn_left()
elif cmd == "d":
    turn_right()
else:
    stop()

Troubleshooting

如果卡住，先看這 4 個地方

第一堂課最常卡在連線、執行和輸入的位置。先查這幾個地方，很多問題都能立刻解決。

找不到 Pico

先確認 USB 線有插好，再請老師一起看 `MicroPython (Raspberry Pi Pico)` 有沒有選對。

按了 Run 沒反應

確認你打開的是 `keyboard-car-control.py`，然後再按一次綠色 `Run`。

打字沒反應

請把游標點到 Thonny 下方 `Shell`，再輸入 `w` 或 `x` 後按 Enter。

車子一直動

先輸入 `x`，如果還是沒有停，立刻請老師協助並先斷開電源。
