你現在要記住的事

  • 每個按鍵都有一個動作。
  • 每個動作後面，通常都會對應一個函式(Function)。
  • 你不用一開始背很多程式，只要先知道「哪個動作對哪個結果」。

def forward(speed=30):
    ...

    ...

    ...

def turn_right(speed=30):
    ...

鍵盤控制(Keyboard Control)

按下按鍵，小車就做動作

cmd = input("Command: ").strip().lower()

if cmd == "w":
    forward()
elif cmd == "s":
    backward()
elif cmd == "a":
    turn_left()
elif cmd == "d":
    turn_right()
else:
    stop()
