Finish Line

本堂完成線

  • 學生知道如何把 Raspberry Pi Pico 小車接上電腦。
  • 學生知道 `W / A / S / D / X` 各自對應的動作。
  • 學生至少成功完成一次前進、停止、轉向。

Not Today

這堂課先不要教

  • 藍牙(Bluetooth) 遙控。
  • 感測器邏輯與循跡。
  • 太多函式細節與硬體延伸。

Before Class

課前先做什麼

先確認課堂目標

  • 今天是入門課，不是完整專案課。
  • 先讓電腦和車子連起來，再教會學生用鍵盤控制基本動作。
  • 不要急著講藍牙(Bluetooth)、感測器或太多延伸功能。

先開這三份就夠了

  1. 學生講義頁
  2. 1 小時授課建議
  3. 小車鍵盤控制範例

In Class

上課時怎麼帶比較順

Part 1

先讓電腦和車子成功連線

先讓學生看到電腦真的能把指令送到車子，這是整堂課最重要的第一步。

Part 2

再讓學生操作五個動作

先把前進、後退、左轉、右轉、停止操作熟，不需要同一堂課再帶更多功能。

Part 3

收在鍵盤控制就好

這堂 Boot Camp 的完成線，就是學生可以用鍵盤控制車子，不必再加感測器與循跡。

After Class

課後可以怎麼延伸

給學生的小延伸

  • 把速度改快或改慢。
  • 設計一段自己的動作路線，例如前進、右轉、停止。
  • 讓學生先猜按鍵結果，再實際操作驗證。

後續才加入的主題

  • 循跡(Line Following)
  • 藍牙遙控(Bluetooth Control)
  • 更多感測器整合
