# microcar

Static course website built from the existing robot-programming teaching materials, especially the lecture decks about the smart oscillating fan project.

## Contents

- `index.html`: main landing page
- `styles.css`: layout, responsive design, and visual system
- `script.js`: reveal-on-scroll interaction
- `assets/`: course-derived visuals used on the site

## Course framing used for the site

- Smart control programming
- Mechanism structure
- Integrated project presentation
- Raspberry Pi Pico, sensors, PWM, Bluetooth, and mechanical transmission

## GitHub Pages

This site is ready to publish as a simple static website.

1. Push the repository to GitHub.
2. In repository settings, open **Pages**.
3. Set the source to **Deploy from a branch**.
4. Choose the `main` branch and the root folder.
5. Save, then wait for GitHub Pages to publish the site.
