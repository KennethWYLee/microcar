20-step Route

學生請照著下面 20 個步驟，從上往下完成

這一區就是本堂課的正式操作流程。左邊看文字，右邊看對應動圖，做完一列再往下走。

STEP 1

先把小車放好

把小車放在平坦桌面或地面，避免一開始就撞出去。

STEP 2

確認桌面空間

確認電腦、滑鼠和小車都在自己看得到的位置，操作時不要被線材卡住。

STEP 3

接上 USB 線

把 USB 線接上 Raspberry Pi Pico 小車與電腦。

STEP 4

等老師確認連線成功

先等老師確認每一組都連線成功，再一起往下操作。

STEP 5

打開 Thonny

在電腦上打開 `Thonny`，準備開始今天的程式操作。

STEP 6

先看右下角解譯器

確認右下角是否為 `MicroPython (Raspberry Pi Pico)`。

STEP 7

如果不是，就跟老師一起切換

若解譯器不是 Pico，先跟著老師一起切換，不要自己亂按其他設定。

STEP 8

建立新的 Python 檔

按 `File > New`，建立一個新的 `.py` 檔案。

STEP 9

往下滑到完整程式區

找到本頁稍後的「完整程式」區塊。

STEP 10

按下複製完整程式

按下「複製完整程式」，把今天要用的程式先複製起來。

STEP 11

回到 Thonny 貼上

切回 Thonny，把整份程式貼到剛剛建立的新檔裡。

STEP 12

先存檔

按 `Ctrl + S`，將檔案存成 `keyboard_car.py`。

STEP 13

按下綠色 Run

按綠色 `Run` 執行程式，讓小車開始等待指令。

STEP 14

把滑鼠點到 Shell

將滑鼠點到下方 `Shell`，接下來所有指令都在這裡輸入。

STEP 15

先輸入 x

先輸入 `x` 後按 Enter，確認知道怎麼停車，這是最安全的開始。

STEP 16

再測試 w、a、d、s

依序輸入 `w`、`a`、`d`、`s` 測試小車前進、左轉、右轉與後退。

STEP 17

每次動完先停車

做完一個動作後，記得先按 `x` 停車，再換下一個指令。

STEP 18

完成 w -> x

先完成 `w -> x`，確認前進與停止都成功。

STEP 19

完成轉向挑戰

完成 `w -> a -> x` 或 `w -> d -> x`，練習方向控制。

STEP 20

說出 5 個按鍵的意思

最後能說出 `W / A / S / D / X` 各自代表的動作，就完成今天的課。

Step 1

先看匯入和腳位

這一段會先告訴小車要使用哪些模組，還有馬達接在哪些腳位上。

from machine import Pin
import sys
import time

M1_A = Pin(12, Pin.OUT)
M1_B = Pin(13, Pin.OUT)
M2_A = Pin(10, Pin.OUT)
M2_B = Pin(11, Pin.OUT)
LED = Pin(25, Pin.OUT)

Step 2

再看前進和停止函式

先記住 `stop()` 和 `forward()`。這兩個函式是今天最重要的基本動作。

def _set_motor(pin_a, pin_b, direction):
    if direction > 0:
        pin_a.value(1)
        pin_b.value(0)
    elif direction < 0:
        pin_a.value(0)
        pin_b.value(1)
    else:
        pin_a.value(0)
        pin_b.value(0)

def stop():
    _set_motor(M1_A, M1_B, 0)
    _set_motor(M2_A, M2_B, 0)

def forward():
    _set_motor(M1_A, M1_B, 1)
    _set_motor(M2_A, M2_B, 1)

Step 3

接著看後退和轉彎

這一段把後退、左轉、右轉都準備好，等等按鍵就會對到這些動作。

def backward():
    _set_motor(M1_A, M1_B, -1)
    _set_motor(M2_A, M2_B, -1)

def turn_left():
    _set_motor(M1_A, M1_B, -1)
    _set_motor(M2_A, M2_B, 1)

def turn_right():
    _set_motor(M1_A, M1_B, 1)
    _set_motor(M2_A, M2_B, -1)

Step 4

再看怎麼讀取指令

這一段會一直等待你在 `Thonny Shell` 輸入的字母。

print("請在 Thonny Shell 輸入 w / a / s / d / x 後按 Enter")

while True:
    cmd = sys.stdin.read(1)
    if not cmd:
        continue

    cmd = cmd.strip().lower()
    if not cmd:
        continue

Step 5

最後看按鍵對應動作

這裡會把輸入的按鍵和對應動作接起來，所以 `w`、`a`、`s`、`d`、`x` 才會真的讓車子動。

    if cmd == "w":
        forward()
    elif cmd == "s":
        backward()
    elif cmd == "a":
        turn_left()
    elif cmd == "d":
        turn_right()
    else:
        stop()
