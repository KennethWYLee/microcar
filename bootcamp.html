Boot Camp Teaching Page

1 小時 Boot Camp 教學頁

這一頁是給老師快速開課用的完整版本。你可以直接依照這裡的節奏帶完一堂 60 分鐘的入門課， 不需要先翻很多檔案再自己組裝。

先下載教學建議 看教師完整頁 看學生任務頁
Boot Camp Goal

這堂課要完成什麼

  • 學生知道小車如何前進、後退、左轉、右轉、停止。
  • 學生知道感測器的數值可以拿來做判斷。
  • 學生能改一個簡單參數，看出差異。

Teaching Flow

60 分鐘建議節奏

如果時間真的只剩 1 小時，建議先求學生看得懂和做得出來，不求一次講到最深。

0-10 分鐘

開場與認識硬體

  • 介紹小車、控制板(Control Board)、電池(Battery)、感測器(Sensor)。
  • 提醒今天的目標是讓小車動起來，不是背全部零件。
  • 先做安全提醒與桌面空間整理。
10-25 分鐘

五個基本動作

  • 示範 `W / A / S / D / X`。
  • 讓學生先把動作和方向連起來。
  • 建議搭配 `小車鍵盤控制範例(Keyboard Control)`。
25-40 分鐘

感測器與 if 判斷

  • 先講「先讀值，再判斷，再反應」。
  • 介紹 `if / elif / else` 的基本概念。
  • 建議搭配 `距離感測範例(Distance Sensor)`。
40-60 分鐘

組合成第一個互動效果

  • 讓學生知道數值不同，反應也會不同。
  • 鼓勵學生修改一個參數。
  • 最後請學生說明他改了什麼、發生了什麼。

Teacher Tips

老師現場可以怎麼問

讓學生說出原因

  • 這一行程式會讓車子做什麼動作？
  • 為什麼距離太近時要停下來或變色？
  • 如果把 10 改成 20，結果會怎麼變？

第一堂課先不要做的事

  • 不要一次講完藍牙(Bluetooth)與所有遙控方式。
  • 不要把硬體接線細節講得太重。
  • 不要讓學生一開始就看完整大專案。
