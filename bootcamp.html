60-min Route

把 1 小時課程拆成 4 個明確階段

以下流程可作為第一堂課的時間安排參考，每一階段都對應明確任務與完成標準。

01 0-10 分鐘

連線成功

先讓每一組都把電腦和小車接好，這是整堂課最重要的起點。

  • 老師做：示範 USB 線和 Thonny 的位置。
  • 學生做：確認自己的車有成功連上。
  • 完成線：知道等一下要在哪裡輸入指令。
02 10-20 分鐘

老師示範

先由教師示範輸入指令，建立按鍵與小車動作的對應關係。

  • 老師做：在 Thonny Shell 輸入 `w / a / s / d / x`。
  • 學生做：口頭說出每個字母控制什麼。
  • 完成線：知道 `x` 是停止鍵。
03 20-40 分鐘

輪流操作

每個學生都要真的摸到一次操作，不只是看老師演示。

  • 老師做：提醒每做完一個動作先停車。
  • 學生做：依序試 `w`、`a`、`s`、`d`、`x`。
  • 完成線：每個人都能讓車子成功動一次。
04 40-60 分鐘

完成任務

最後用一個簡單小挑戰收尾，讓學生有完成感。

  • 老師做：出一段簡單動作題。
  • 學生做：完成「前進、左轉、停止」或「前進、右轉、後退、停止」。
  • 完成線：能說出 `W / A / S / D / X` 對應的動作。

Thonny Steps

先照這個順序操作 Thonny

依照以下順序逐步操作，即可完成本堂課的基本連線與控制。

Step 1. 接上 USB 線

  • 把 Raspberry Pi Pico 小車接到電腦。
  • 請保持車子與電腦連線，不要任意拔除接線。

Step 2. 打開 Thonny

  • 右下角確認解譯器是 `MicroPython (Raspberry Pi Pico)`。
  • 如果還沒選到，請跟著老師一起設定。

Step 3. 開啟今天的程式

  • 下載 `keyboard-car-control.py`。
  • 在 Thonny 打開這份檔案。
  • 按綠色 `Run` 執行。

Step 4. 到 Shell 打字控制

  • 在 Thonny 下方 `Shell` 輸入 `w` 後按 Enter。
  • 要停車時輸入 `x` 後按 Enter。
  • 如果車子不聽話，先輸入 `x`。

Teaching Sequence

建議教學順序

課堂投影可依照以下順序進行，每一步聚焦一個重點，較有利於學生跟上操作。

Step A

先講操作，不先講理論

先完成操作示範，再回頭說明函式與指令，可提高學習動機與理解度。

Step B

先讀主程式的前半段

建議先閱讀匯入模組、設定腳位，以及 `stop()` 和 `forward()` 相關內容。

Step C

最後才看按鍵對應

學生先理解小車動作後，再閱讀 `if cmd == "w"` 這段，較容易建立對應關係。

Part 1

先匯入模組，設定腳位

本段設定控制小車所需的腳位與模組。

from machine import Pin
import sys
import time

M1_A = Pin(12, Pin.OUT)
M1_B = Pin(13, Pin.OUT)
M2_A = Pin(10, Pin.OUT)
M2_B = Pin(11, Pin.OUT)
LED = Pin(25, Pin.OUT)

Part 2

先做出「停車」和「往前」

本段可先聚焦 `stop()` 與 `forward()`，理解函式如何控制小車動作。

def _set_motor(pin_a, pin_b, direction):
    if direction > 0:
        pin_a.value(1)
        pin_b.value(0)
    elif direction < 0:
        pin_a.value(0)
        pin_b.value(1)
    else:
        pin_a.value(0)
        pin_b.value(0)

def stop():
    _set_motor(M1_A, M1_B, 0)
    _set_motor(M2_A, M2_B, 0)

def forward():
    _set_motor(M1_A, M1_B, 1)
    _set_motor(M2_A, M2_B, 1)

Part 3

再補齊後退、左轉、右轉

本段補齊後退、左轉與右轉，構成小車的基本動作集合。

def backward():
    _set_motor(M1_A, M1_B, -1)
    _set_motor(M2_A, M2_B, -1)

def turn_left():
    _set_motor(M1_A, M1_B, -1)
    _set_motor(M2_A, M2_B, 1)

def turn_right():
    _set_motor(M1_A, M1_B, 1)
    _set_motor(M2_A, M2_B, -1)

Part 4

讀取鍵盤指令

本段負責接收 `Thonny Shell` 輸入的單一指令。

print("請在 Thonny Shell 輸入 w / a / s / d / x 後按 Enter")

while True:
    cmd = sys.stdin.read(1)
    if not cmd:
        continue

    cmd = cmd.strip().lower()
    if not cmd:
        continue

Part 5

把按鍵對應到動作

本段將每個按鍵對應到一個動作函式，是本堂課的核心控制邏輯。

    if cmd == "w":
        forward()
    elif cmd == "s":
        backward()
    elif cmd == "a":
        turn_left()
    elif cmd == "d":
        turn_right()
    else:
        stop()

Practice Order

實際操作時，建議這樣練

先從最安全、最容易成功的開始。不要一開始就連續按很多指令。

練習 1

先試 `x`

先知道怎麼停車，這樣等一下比較安心。

練習 2

再試 `w` 然後 `x`

先讓車子前進一下，再停下來。

練習 3

加入 `a` 和 `d`

讓車子左轉、右轉，熟悉方向感。

練習 4

最後試 `s`

讓車子後退，再記得停車。
