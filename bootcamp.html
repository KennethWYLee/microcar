Boot Camp Teaching Page

1 小時 Boot Camp 教學頁

這一頁是給你教國中生用的 60 分鐘版本。這堂課不追求循跡，也不先碰感測器， 只先完成一件最重要的事：讓電腦和 Raspberry Pi Pico 小車成功串聯， 然後用電腦打字控制車子的前進、後退、左轉、右轉、停止。

先下載教學建議 看教師完整頁 看學生任務頁
Boot Camp Goal

這堂課要完成什麼

  • 學生知道如何把電腦和 Raspberry Pi Pico 小車連起來。
  • 學生知道鍵盤輸入可以控制車子動作。
  • 學生能用 `W / A / S / D / X` 控制前進、左轉、後退、右轉、停止。

Teaching Flow

60 分鐘建議節奏

如果時間真的只剩 1 小時，建議只做最穩定、最有成就感的版本，不再加感測器與循跡。

0-10 分鐘

連接電腦和小車

  • 介紹 Raspberry Pi Pico 小車、USB 連線與電腦端操作方式。
  • 確認學生知道哪裡插線、哪裡打字、哪裡會看到回應。
  • 提醒今天的目標是「成功控制車子」，不是一次學完整個專案。
10-20 分鐘

示範打字控制

  • 示範在電腦端輸入 `W / A / S / D / X`。
  • 讓學生知道一個字元就能對應一個車子動作。
  • 全程建議搭配 `小車鍵盤控制範例(Keyboard Control)`。
20-40 分鐘

學生輪流操作五個動作

  • 先讓學生輪流操作前進、後退、左轉、右轉、停止。
  • 讓學生把鍵盤字母和實際動作連起來。
  • 這一段重點是熟悉操作，不需要加入更多功能。
40-60 分鐘

收尾與挑戰

  • 可以請學生挑戰連續輸入一串動作，像是前進、左轉、停止。
  • 最後請學生口頭說出每個按鍵控制什麼。
  • 如果時間夠，再簡單介紹「之後還能學到循跡和感測器」。

Teacher Tips

老師現場可以怎麼問

讓學生說出原因

  • 如果我打 `W`，車子會做什麼？
  • 如果我打 `A` 或 `D`，車子會往哪邊轉？
  • 為什麼 `X` 很重要？它代表什麼？

第一堂課先不要做的事

  • 不要一次講完藍牙(Bluetooth)與所有遙控方式。
  • 不要把感測器、循跡、if 判斷都塞進同一堂課。
  • 不要讓學生一開始就看完整大專案。
