60-min Route

1 小時可以怎麼學

今天的版本先求穩定和成功感。你只要會控制小車，就已經完成這堂課最重要的目標。

0-10 分鐘

先把電腦和小車連起來

  • 把 USB 線接好。
  • 打開 `Thonny`。
  • 確認老師的車和你的車都能被電腦看到。
10-20 分鐘

先看老師示範一次

  • 看老師在 `Thonny Shell` 輸入 `w / a / s / d / x`。
  • 看懂每個字母對應什麼動作。
  • 先記住 `x` 是停止。
20-40 分鐘

換你自己操作

  • 先試 `w` 和 `x`。
  • 再試 `a`、`d` 和 `s`。
  • 每做完一個動作，都記得讓小車停下來。
40-60 分鐘

完成小挑戰

  • 讓小車完成「前進、左轉、停止」。
  • 讓小車完成「前進、右轉、後退、停止」。
  • 最後說出每個字母控制什麼動作。

Thonny Steps

先照這個順序操作 Thonny

你現在不用自己記全部，只要一個步驟一個步驟跟著做就可以。

Step 1. 接上 USB 線

  • 把 Raspberry Pi Pico 小車接到電腦。
  • 先不要亂拔線，讓車子和電腦保持連線。

Step 2. 打開 Thonny

  • 右下角確認解譯器是 `MicroPython (Raspberry Pi Pico)`。
  • 如果還沒選到，請跟著老師一起設定。

Step 3. 開啟今天的程式

  • 下載 `keyboard-car-control.py`。
  • 在 Thonny 打開這份檔案。
  • 按綠色 `Run` 執行。

Step 4. 到 Shell 打字控制

  • 在 Thonny 下方 `Shell` 輸入 `w` 後按 Enter。
  • 要停車時輸入 `x` 後按 Enter。
  • 如果車子不聽話，先輸入 `x`。

Code Walkthrough

今天的程式，從上到下這樣看

你不用一次看整份。跟著老師從上往下看，就會知道每一段在做什麼。

Part 1

先匯入模組，設定腳位

這一段是在告訴小車：「等一下我要控制哪些腳位。」

from machine import Pin
import sys
import time

M1_A = Pin(12, Pin.OUT)
M1_B = Pin(13, Pin.OUT)
M2_A = Pin(10, Pin.OUT)
M2_B = Pin(11, Pin.OUT)
LED = Pin(25, Pin.OUT)

Part 2

先做出「停車」和「往前」

老師可以先帶你看 `stop()` 和 `forward()`，知道函式是拿來控制動作的。

def _set_motor(pin_a, pin_b, direction):
    if direction > 0:
        pin_a.value(1)
        pin_b.value(0)
    elif direction < 0:
        pin_a.value(0)
        pin_b.value(1)
    else:
        pin_a.value(0)
        pin_b.value(0)

def stop():
    _set_motor(M1_A, M1_B, 0)
    _set_motor(M2_A, M2_B, 0)

def forward():
    _set_motor(M1_A, M1_B, 1)
    _set_motor(M2_A, M2_B, 1)

Part 3

再補齊後退、左轉、右轉

當你看得懂方向怎麼改，小車的基本動作就差不多都完成了。

def backward():
    _set_motor(M1_A, M1_B, -1)
    _set_motor(M2_A, M2_B, -1)

def turn_left():
    _set_motor(M1_A, M1_B, -1)
    _set_motor(M2_A, M2_B, 1)

def turn_right():
    _set_motor(M1_A, M1_B, 1)
    _set_motor(M2_A, M2_B, -1)

Part 4

讀取鍵盤指令

這一段是在等待你從 `Thonny Shell` 輸入一個字母。

print("請在 Thonny Shell 輸入 w / a / s / d / x 後按 Enter")

while True:
    cmd = sys.stdin.read(1)
    if not cmd:
        continue

    cmd = cmd.strip().lower()
    if not cmd:
        continue

Part 5

把按鍵對應到動作

這是今天最重要的一段。每個字母都會呼叫一個動作函式。

    if cmd == "w":
        forward()
    elif cmd == "s":
        backward()
    elif cmd == "a":
        turn_left()
    elif cmd == "d":
        turn_right()
    else:
        stop()

Practice Order

實際操作時，建議這樣練

先從最安全、最容易成功的開始。不要一開始就連續按很多指令。

練習 1

先試 `x`

先知道怎麼停車，這樣等一下比較安心。

練習 2

再試 `w` 然後 `x`

先讓車子前進一下，再停下來。

練習 3

加入 `a` 和 `d`

讓車子左轉、右轉，熟悉方向感。

練習 4

最後試 `s`

讓車子後退，再記得停車。
