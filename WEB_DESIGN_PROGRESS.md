# 網頁設計進度紀錄

最後更新日期：2026-04-10

## 專案目標

這個網站的目標是把 `機器人` 課程整理成一個適合教學與自學的靜態網站，放在 GitHub repository 上，之後可搭配 GitHub Pages 對外開放。

目前網站定位為：
- 中文為主
- 關鍵字補充英文，格式為 `中文(English)`
- 以小車程式、小車控制、感測器、馬達與機構整合為主題
- 主要對象為國小、國中學生與授課老師

## 目前設計方向

### 視覺方向

目前網站採用：
- 溫暖、清楚、易閱讀的配色
- 卡片式資訊分區
- 插圖與流程圖並重
- 手機與桌機皆可閱讀

### 教學方向

目前網站的核心教學原則是：
- 老師先有清楚的授課脈絡
- 學生先看懂圖，再碰程式
- 用小步驟學習代替一次塞太多內容
- 讓學生可以從模仿到修改，再慢慢建立程式思維
