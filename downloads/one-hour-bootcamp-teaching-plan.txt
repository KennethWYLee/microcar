Raspberry Pi Pico 小車 1 小時 Boot Camp 教學建議

適用對象
- 國中生為主
- 國小高年級可在老師帶領下使用

課程目標
- 讓電腦和 Raspberry Pi Pico 小車成功連線
- 用電腦打字控制車子
- 完成前進、後退、左轉、右轉、停止

建議時間安排
1. 0-10 分鐘
   連接電腦和車子，確認操作位置

2. 10-20 分鐘
   示範鍵盤輸入和五個基本動作

3. 20-40 分鐘
   學生輪流實作前進、後退、左轉、右轉、停止

4. 40-60 分鐘
   進行簡單挑戰，例如連續輸入一串動作

第一堂課先不要放進來
- 藍牙遙控
- 感測器與 if / elif / else
- 循跡邏輯
- 過長的完整專案程式

老師可用的提問
- 如果我打 W，車子會怎麼動？
- 如果我打 A，車子會往哪邊轉？
- X 這個指令為什麼重要？

學生完成後的最小成果
- 能把電腦和車子連起來
- 能用鍵盤打字控制車子
- 能說出 W / A / S / D / X 分別代表什麼
