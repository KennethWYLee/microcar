Raspberry Pi Pico 小車 1 hr Boot Camp 教學建議

課程定位
- 對象：國中生
- 目標：在 1 小時內完成電腦與小車連線，並用 Thonny 控制小車做出五個基本動作。

本堂課完成線
- 電腦成功連上 Raspberry Pi Pico 小車。
- 學生能在 Thonny 建立新檔並貼上完整程式。
- 學生能在 Shell 輸入 W / A / S / D / X 控制小車。

建議教學順序
1. 0-10 分鐘
   接上 USB，確認 Pico 小車已連線，打開 Thonny。
2. 10-20 分鐘
   確認 Interpreter 為 MicroPython (Raspberry Pi Pico)，建立新檔。
3. 20-35 分鐘
   從網站複製完整程式，貼到 Thonny，存成 keyboard_car.py。
4. 35-45 分鐘
   按下 Run，到 Shell 先輸入 x，再試 w / a / d / s。
5. 45-60 分鐘
   讓學生輪流操作，最後統整五個按鍵對應的動作。

本堂課先不要放進去的內容
- 藍牙遙控
- 感測器條件判斷
- 循跡邏輯
- 過長的進階專案程式

老師現場提醒
- 小車一開始先放穩，不要直接朝桌邊。
- 先教 x 停止，再教前進與轉向。
- 如果沒有反應，先檢查 USB 與 Interpreter。
