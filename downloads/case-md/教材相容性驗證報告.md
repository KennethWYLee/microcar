## 原始小車硬體規格對照

依 PDF 第 26 頁與相關實作頁面整理：

| 功能 | 教材規格 | 延伸教材使用狀態 |
|---|---:|---|
| 內建 LED | GP25 | 已統一使用 `Pin(25, Pin.OUT)` |
| 按鈕 | GP3，下拉電阻 | 使用 `Pin(3, Pin.IN)`，符合 |
| 蜂鳴器 | GP6 | 使用 `PWM(Pin(6))`，符合 |
| RGB 彩燈條 | GP2，WS2812B，6 顆 | 使用 `WS2812B(pin=2, leds=6)`，符合 |
| 超音波/距離 RGB | Sensor1：RGB GP14、聲波 GP15 | 使用 `RUS04(sensor_pin=15, rgb_pin=14)`，符合 |
| 右馬達 M1 | GP12、GP13 | 使用 `Motor(a_pin=12, b_pin=13)`，符合 |
| 左馬達 M2 | GP11、GP10 | 使用 `Motor(a_pin=11, b_pin=10)`，符合 |
| I2C | SDA GP20、SCL GP21 | 使用 `mango.bus.i2c`，符合函式庫設定 |
| 循跡感測器 | I2C 四路尋跡 | 已調整為 `GraySensor8(..., digital_channels='4')` |
| 類比輸入 A0 | GP26 | ADC case 使用 `ADC(Pin(26))`，符合 |
| 外接 LED D7 | GP7 | PWM LED case 使用 `PWM(Pin(7))`，符合 |
| 伺服 | PDF 範例使用 GP6 | Servo case 使用 `Servo(pin=6)`，符合；但會與蜂鳴器共用腳位 |

## 已修正項目

1. 入門篇原先部分範例使用 `Pin("LED")`，已改成 PDF 指定的 `Pin(25, Pin.OUT)`。
2. 無人車與專題化篇中，循跡控制原先使用 8 路 `digital_channels='ALL'`，已改成符合 PDF 的四路 `digital_channels='4'`。
3. 四路循跡的加權控制已改為 `WEIGHTS = [-2, -1, 1, 2]`，並將 `range(8)` 改為 `range(4)`。
4. 兩段應用題解答原本是片段式補充，已改成可單獨執行的完整程式。
