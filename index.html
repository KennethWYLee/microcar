1 小時 Boot Camp

五月國中生無人小車 Boot Camp 專區。

這一區是為你五月的課先整理好的快速授課頁。目標不是把所有理論一次講完， 而是讓國中生在 1 小時內真的操作無人小車，理解「指令、感測器、動作」三者怎樣連在一起。

Boot Camp Snapshot

一堂讓學生看得到、操作得到、說得出來的 60 分鐘無人小車 Boot Camp。

60 分鐘(60 min) 國中生(Junior High) 2 到 4 人一組(Team of 2-4) Boot Camp 體驗課

建議你把這堂課聚焦在三件事：先讓學生知道無人小車怎樣接收指令， 再讓他們操作一次前進、轉向、停止，最後體驗感測器如何幫小車自動避障(Obstacle Avoidance)。

  • 學生能分辨手動控制(Manual Control) 與自動控制(Autonomous Control) 的差別。
  • 學生能操作小車完成前進、左轉、右轉、停止。
  • 學生能理解感測器偵測到距離後，程式會做出判斷。
  • 學生能用自己的話說出小車為什麼會避開障礙物。

60-min Plan

一小時授課節奏

0-10 分鐘 認識無人小車、說明安全規則、展示今天的挑戰目標。
10-20 分鐘 用圖解介紹前進、後退、左轉、右轉、停止五個基本指令。
20-35 分鐘 學生分組操作小車，完成基本遙控或按鍵控制。
35-50 分鐘 示範感測器避障，讓學生觀察距離判斷如何改變小車動作。
50-60 分鐘 進行小挑戰，例如避開障礙物或走過簡單路線，最後請學生分享。

學習目標(Learning Outcomes)

建議把每節課的學習成果收斂成「看得懂、說得出、改得動」三層。

對國小與國中階段而言，最重要的不只是完成作品，而是能說出一個指令如何變成一個動作，並且願意自己改一個條件或參數。

  • 能說出輸入(Input)、判斷(Condition)、輸出(Output) 的關係。
  • 能辨認 `forward()`、`turn_left()`、`stop()` 這類函式的用途。
  • 能把感測器結果和 `if / elif / else` 對應起來。
  • 能修改一個速度、距離或方向設定，觀察小車反應。

課程定位 Overview

小車與風扇只是載體，真正學的是控制(Control) 與整合(Integration)。

控制思維(Control Thinking)

學生要學會怎樣讓輸入(Input) 經過判斷(Condition) 後，變成輸出(Output) 與動作(Motion)。

程式思維(Programming Thinking)

從函式(Function)、條件判斷(If / Elif)、迴圈(Loop) 到指令映射(Command Mapping) 逐步建立邏輯。

硬體思維(Hardware Thinking)

理解按鈕(Button)、LED、感測器(Sensor)、馬達(Motor) 與控制板(Control Board) 的角色。

作品整合(Project Integration)

最後能把小車與風扇專題串起來，知道「哪個程式控制哪個動作」。

授課建議 Teaching Plan

先讓學生看得懂，再讓學生寫得出來。

老師授課順序(Teacher Flow)

  • 先用圖解說明小車的五個基本指令：前進、後退、左轉、右轉、停止。
  • 再帶學生看程式骨架：初始化(Init) -> 定義函式(Function) -> 接收指令(Input) -> 判斷 -> 輸出。
  • 接著用 LED、按鈕、感測器做簡單練習，建立「輸入控制輸出」的概念。
  • 最後再帶入馬達控制(Motor Control) 與藍牙遙控(Bluetooth Control)。

學生自學順序(Self-study Flow)

  • 先看圖解學習區，把動作和程式流程看懂。
  • 再照著程式練習區打一次程式，不要只是看。
  • 接著修改速度、條件或方向，觀察程式改變後小車的反應。
  • 最後再回到下載區看簡報與 PDF，補齊細節。

最適合的節奏(Class Rhythm)

建議用「圖解 5 分鐘 -> 範例程式 10 分鐘 -> 實作 10 分鐘 -> 回饋 5 分鐘」的節奏。對初學者來說，先看懂圖，再寫程式，吸收會快很多。

LED 基礎練習(LED Basics)

LED.py

建立最基本的輸出(Output) 概念。

from machine import Pin
import time

led = Pin("LED", Pin.OUT)

while True:
    led.toggle()
    time.sleep(1)

按鈕控制 LED(Button to LED)

btn control led.py

理解輸入(Input) 怎樣控制輸出(Output)。

from machine import Pin
from time import sleep

led = Pin(25, Pin.OUT)
btn = Pin(3, Pin.IN)

while True:
    led.value(btn.value())
    sleep(0.3)

感測器條件判斷(Sensor Condition)

change sensor color by dist.py

這段對應上面的感測器圖解，非常適合初學者理解 if / elif。

dist = sensor.ping()

if dist < 10:
    sensor.rgb_all(utils.COLOR_RED)
elif dist < 20:
    sensor.rgb_all(utils.COLOR_BLUE)
else:
    sensor.rgb_all(utils.COLOR_GREEN)

def forward():
