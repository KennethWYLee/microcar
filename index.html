Code patterns

Representative examples from the course files.

LED Warm-up

LED.py 
from machine import Pin
import time

led = Pin("LED", Pin.OUT)

while True:
    led.toggle()
    time.sleep(1)

Button to Actuator Logic

btn control led.py 
from machine import Pin
from time import sleep

led = Pin(25, Pin.OUT)
btn = Pin(3, Pin.IN)

while True:
    led.value(btn.value())
    sleep(0.3)

Distance-driven RGB Response

change sensor color by dist.py 
dist = sensor.ping()

if dist < 10:
    sensor.rgb_all(utils.COLOR_RED)
elif dist < 20:
    sensor.rgb_all(utils.COLOR_BLUE)
else:
    sensor.rgb_all(utils.COLOR_GREEN)
