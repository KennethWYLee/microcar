課程定位 Overview

這門課不是只做電風扇，而是在教完整的機器人專題思維。

主題作品(Project Theme)

以智慧型可擺動調速電風扇作為專題，讓學生從作品中理解控制與結構。

控制能力(Control)

學習數位輸入輸出(Digital Input/Output)、類比輸出(PWM)、感測器(Sensor)與藍牙遙控(Bluetooth Control)。

機構能力(Mechanism)

理解齒輪傳動(Gear Transmission)、蝸桿蝸輪(Worm Gear)、連桿機構(Linkage Mechanism)與擺頭運動。

整合能力(Integration)

把電路、程式、感測與機構整合成可運作的實體作品，讓學生真的能解釋作品如何運作。

授課建議 Teaching Plan

老師可以怎麼用這個網站上課，學生又可以怎麼自學。

老師授課建議(Teacher Flow)

  • 先帶學生看控制板(Control Board)與材料清單，建立整體印象。
  • 接著先做最簡單的 LED 與按鈕(Button) 程式，建立成功經驗。
  • 再進入馬達(Motor)、PWM、感測器(Sensor) 與顯示(Display) 應用。
  • 最後再講機構(Mechanism) 與組裝，學生會更容易理解作品為什麼會動。

學生自學建議(Self-study Flow)

  • 先看「單元地圖(Lesson Map)」，知道每一節在學什麼。
  • 再看「材料設備(Build Kit)」，認識每個元件的功能。
  • 接著從程式練習(Code Practice) 依序操作，不要一開始就做整合專題。
  • 最後回到下載區(Downloads) 把簡報與 PDF 打開對照複習。

課堂節奏建議(Class Rhythm)

推薦節奏是「概念 5 分鐘 -> 接線 5 分鐘 -> 程式 10 分鐘 -> 展示 5 分鐘 -> 提問 5 分鐘」，讓學生每一小段都有看見、做出來、再理解的機會。

控制板(Control Board)

負責電源、感測器、顯示器與輸出控制，是整門課的核心平台。

機構組件(Mechanical Structure)

用來說明擺頭運動、傳動減速與結構組裝，讓學生理解為什麼作品可以左右擺動。

感測器(Sensor)

學生可透過感測值(Sensing Data) 練習互動控制與智慧反應。

輸入模組(Input Module)

適合搭配條件判斷(Condition) 與邏輯控制(Control Logic) 教學。

機構概念 Mechanism

把原本比較分散的機構內容，整理成三個好講的重點。

機構(Mechanism)

由兩個以上機件(Machine Parts) 組成，透過特定連結關係產生預期的運動。

齒輪傳動(Gear Transmission)

負責改變方向(Direction Change) 與減速(Speed Reduction)，讓馬達輸出更適合帶動擺頭。

四連桿(Four-bar Linkage)

把旋轉運動(Rotary Motion) 轉成搖擺運動(Oscillating Motion)，這就是風扇擺頭的核心原理。

程式練習 Code Practice

這一區老師可以直接示範，學生也可以照著做。

LED 基礎練習(LED Basics)

LED.py

最適合作為第一個成功案例，讓學生先看到控制輸出(Output) 的效果。

from machine import Pin
import time

led = Pin("LED", Pin.OUT)

while True:
    led.toggle()
    time.sleep(1)

按鈕控制 LED(Button to LED)

btn control led.py

從按鈕(Button) 到 LED 的互動，是理解數位輸入輸出(Digital I/O) 最直覺的一步。

from machine import Pin
from time import sleep

led = Pin(25, Pin.OUT)
btn = Pin(3, Pin.IN)

while True:
    led.value(btn.value())
    sleep(0.3)

距離感測與顏色回饋(Distance to RGB)

change sensor color by dist.py

這段非常適合教條件判斷(Condition) 與互動式回饋(Interactive Feedback)。

dist = sensor.ping()

if dist < 10:
    sensor.rgb_all(utils.COLOR_RED)
elif dist < 20:
    sensor.rgb_all(utils.COLOR_BLUE)
else:
    sensor.rgb_all(utils.COLOR_GREEN)

馬達方向控制(Motor Direction)

moto.py

學生只要理解這一段，就更容易把程式和擺頭作品的動作連起來。

def forward():
    _set_motor(M1_A, M1_B, 1)
    _set_motor(M2_A, M2_B, 1)

def turn_left():
    _set_motor(M1_A, M1_B, -1)
    _set_motor(M2_A, M2_B, 1)
