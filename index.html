1 小時 Boot Camp

60 分鐘內，帶學生完成第一台能連線、能用鍵盤控制的 Raspberry Pi Pico 小車。

這是一堂偏「入門體驗 + 基本操作」的課，不追求一次教完所有功能。 第一堂課的重點只有兩件事：連線成功，以及用鍵盤控制前進、後退、左轉、右轉、停止。

進入完整 Boot Camp 頁
Boot Camp Snapshot

課程定位

  • 對象：國中生(Junior High)為主，也適合國小高年級。
  • 形式：2 到 4 人一組，邊看邊操作。
  • 目標：完成小車基本移動與鍵盤控制。
  • 延伸：循跡(Line Following)、感測器(Sensor)與藍牙(Bluetooth)都放在後續單元。
60-min Plan

建議節奏

0-10 分鐘 把電腦和 Raspberry Pi Pico 小車連起來，確認操作位置與安全提醒。
10-20 分鐘 示範 `W / A / S / D / X` 和前進、後退、左轉、右轉、停止五個基本動作。
20-40 分鐘 讓學生輪流操作五個按鍵，確認每個人都能成功控制小車。
40-60 分鐘 做簡單任務，例如前進、左轉、右轉、停止，然後口頭回顧今天學到的指令。

鍵盤控制(Keyboard Control)

先讀按鍵，再做動作

這段程式最重要的是順序：先讀到按鍵，再決定小車要做什麼。這就是「指令對應動作」的概念。

cmd = input("Command: ").strip().lower()

if cmd == "w":
    forward()
elif cmd == "s":
    backward()
elif cmd == "a":
    turn_left()
elif cmd == "d":
    turn_right()
else:
    stop()
