Quick Start

1 hr Boot Camp 保留為快速體驗課

Boot Camp 是一小時活動入口，適合先讓學生用 Thonny 連上 Raspberry Pi Pico 小車，貼上程式並用鍵盤控制基本動作。它會保留在網站中，但不併入 5 個正式教材主題。

1 hr Boot Camp

先完成連線與基本控制

學生照著 12 個步驟操作：開啟 Thonny、建立新檔、貼上完整程式、按 Run，最後用 W / A / S / D / X 控制小車。
