1 小時 Boot Camp

五月國中生無人小車 60 分鐘體驗課。

這一區專門給五月的課使用。目標不是一次講完所有理論，而是讓學生在一小時內真的看到、 操作到、說得出小車如何接收指令、判斷環境，再做出動作。

Boot Camp Snapshot

一堂讓學生操作無人小車、理解控制流程的 60 分鐘入門課。

60 分鐘(60 min) 國中生(Junior High) 2 到 4 人一組(Team of 2-4) Hands-on Demo

建議把這堂課聚焦在三件事：認識指令、操作小車、理解感測器避障。 只要學生能說出「偵測到障礙物後，小車為什麼改變方向」，這堂課就很成功。

  • 學生能分辨手動控制(Manual Control) 與自動控制(Autonomous Control)。
  • 學生能完成前進、左轉、右轉、停止等基本操作。
  • 學生能理解感測器偵測距離後，程式會進行條件判斷。
  • 學生能用自己的話說出小車為什麼會避障。
60-min Plan

一小時授課節奏

0-10 分鐘 認識無人小車、說明安全規則、展示今天的任務。
10-20 分鐘 介紹前進、後退、左轉、右轉、停止等基本指令。
20-35 分鐘 學生分組操作小車，完成基本動作控制。
35-50 分鐘 示範感測器避障，觀察偵測與動作的關係。
50-60 分鐘 完成小挑戰，最後請學生分享他看到的控制邏輯。
Teacher Checklist

上課前先備好

小車本體(Car) 控制板(Control Board) 電池(Battery) 感測器(Sensor) 障礙物(Props) 投影片(Slides)

如果時間只有一小時，建議硬體先準備完成，課堂上把時間留給操作、觀察和分享。

Student Missions

國中生 Boot Camp 任務

任務 A 成功讓小車前進、轉彎、停止。
任務 B 看懂感測器遇到障礙物時，小車為什麼會改變方向。
任務 C 完成一個小挑戰，例如繞過障礙物或走到指定終點。

教師區 Teacher Zone

把授課節奏、教學重點和教材使用方式收在同一區。

教師區保留較專業的授課視角，但不再拆成很多不同區塊。老師只要看這一區，就能快速知道怎麼帶課、怎麼安排學生任務。

Learning Outcomes

建議把學習成果收斂成「看得懂、說得出、改得動」三層。

對國中生而言，最重要的是能說出指令怎樣變成動作，並且願意親手改一個條件或參數。

  • 能說出輸入(Input)、判斷(Condition)、輸出(Output) 的關係。
  • 能辨認前進、轉向、停止等動作函式的用途。
  • 能把感測器結果和 `if / elif / else` 對應起來。
  • 能修改一個速度、距離或方向設定，觀察小車反應。
Teaching Route

建議 4 段授課地圖

指令與動作 先用圖解建立基本方向感與控制概念。
程式與流程 再介紹最基本的函式、輸入與條件判斷。
感測與反應 讓學生看到感測器資料如何影響小車動作。
挑戰與分享 用小任務檢查學生是否真正看懂。
Classroom Tips

一堂課內最值得保留的做法

先示範再放手

先做一次完整示範，再讓學生自己操作，會比直接講語法更有效。

一次只教一個重點

先讓學生理解一種控制邏輯，再往下帶感測器和自動避障。

Assessment

可直接觀察的學習表現

口語解釋 學生能說明按鍵、感測器和小車動作的關係。
程式修改 學生能改一個數字或條件，並看懂結果。
實機展示 學生能完成基本動作或簡單避障挑戰。

LED 基礎練習(LED Basics)

LED.py

先建立最基本的輸出(Output) 概念。

from machine import Pin
import time

led = Pin("LED", Pin.OUT)

while True:
    led.toggle()
    time.sleep(1)

感測器條件判斷(Sensor Condition)

change sensor color by dist.py

這段最適合搭配上面的感測器圖解一起看。

dist = sensor.ping()

if dist < 10:
    sensor.rgb_all(utils.COLOR_RED)
elif dist < 20:
    sensor.rgb_all(utils.COLOR_BLUE)
else:
    sensor.rgb_all(utils.COLOR_GREEN)

小車動作函式(Car Motion Functions)

moto.py

把「前進」和「左轉」變成函式，程式會更清楚。

def forward():
    _set_motor(M1_A, M1_B, 1)
    _set_motor(M2_A, M2_B, 1)

def turn_left():
    _set_motor(M1_A, M1_B, -1)
    _set_motor(M2_A, M2_B, 1)
