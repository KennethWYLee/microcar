Core Tool

第一個主題先學會用 Thonny 操作小車

這套課程以 Thonny 作為主要開發工具。 學生只要能開新檔、貼上程式、按下 Run、回到 Shell 輸入指令，就可以完成第一個主題。

Main Tool

Thonny 是第一堂課最重要的工具

Boot Camp 不需要先下載新的程式檔，學生直接把網站上的程式貼到 Thonny，就能讓小車動起來。

  • 開啟 Thonny。
  • 確認直譯器為 MicroPython (Raspberry Pi Pico)。
  • 建立新檔，貼上網站上的完整程式。
  • 按下 Run，到 Shell 輸入指令。
