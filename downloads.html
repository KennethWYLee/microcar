Setup Tool

開始之前，先準備好 Thonny

本課以 `Thonny` 作為主要操作工具。若教室電腦尚未安裝，建議先完成安裝，再進入下方教材與程式內容。

Official Site

先到 Thonny 官網

官方網站首頁就有 Windows 下載入口，也能看到這個工具本身主打的是給初學者使用的 Python IDE。

前往 thonny.org 看課堂操作順序
In Class

安裝好後，今天只做這些

  • 打開 Thonny。
  • 確認解譯器是 `MicroPython (Raspberry Pi Pico)`。
  • 到 Boot Camp 頁複製完整程式，貼到 Thonny 後按 `Run`。
