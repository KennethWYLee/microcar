Key Code

用兩段程式把板子概念接起來

第一段讓學生理解輸入與輸出，第二段讓老師銜接到馬達控制邏輯。

LED 與按鈕：輸入輸出最小範例
from machine import Pin
import time

led = Pin(25, Pin.OUT)
button = Pin(3, Pin.IN)

while True:
    if button.value() == 1:
        led.on()
    else:
        led.off()
    time.sleep(0.05)
馬達方向：兩個腳位決定一個馬達狀態
# M1 右馬達：GP12 / GP13
# M2 左馬達：GP11 / GP10

def set_motor(pin_a, pin_b, direction):
    if direction > 0:
        pin_a.value(1)
        pin_b.value(0)
    elif direction < 0:
        pin_a.value(0)
        pin_b.value(1)
    else:
        pin_a.value(0)
        pin_b.value(0)

How To Use

這份 PDF 在網站中的角色

課前 先用腳位表確認硬體

確認課堂要用的 LED、按鈕、蜂鳴器、感測器或馬達是否與主板腳位一致。

課中 讓學生從一個元件開始

先做 LED 或按鈕，再逐步銜接蜂鳴器、RGB、感測器與馬達。

除錯 先查腳位，再查程式

如果程式沒反應，優先檢查腳位、接線與元件方向，再檢查程式邏輯。

延伸 把元件組合成任務

學生熟悉單一元件後，再把輸入、輸出、馬達與感測器組成小作品。
