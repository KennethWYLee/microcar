Quick Goals

今天的 3 個小目標

Goal 1

接好車子

把 Raspberry Pi Pico 小車接上電腦，確認老師示範時看得到自己的車。

Goal 2

看懂 Thonny

知道要在哪裡開檔、按 Run、在哪裡輸入指令。

Goal 3

控制小車

會用 `W / A / S / D / X` 控制前進、後退、左轉、右轉、停止。

Keyboard Map

先背這 5 個按鍵就夠了

Keyboard Control
W前進
A左轉
S後退
D右轉
X停止

記住這 3 件事

  • 每個按鍵都對應一個動作。
  • 每做完一個動作，最好先按 `X` 停車。
  • 先會操作，不用急著把程式全部背起來。

Need Help?

卡住時，先看這 3 件事

找不到 Pico

先確認 USB 線有插好，再請老師一起看解譯器有沒有選對。

按了 Run 沒反應

先看是不是打開 `keyboard-car-control.py`，再按一次綠色 `Run`。

打字沒反應

把游標點到下方 `Shell`，輸入 `w` 或 `x` 後再按 Enter。
