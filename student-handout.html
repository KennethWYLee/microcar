Keyboard Map

先熟悉這 5 個按鍵

Keyboard Control
W前進
A左轉
S後退
D右轉
X停止

記住這 3 件事

  • 每個按鍵都對應一個動作。
  • 每做完一個動作，最好先按 `X` 停車。
  • 先熟悉操作，再逐步理解完整程式。

Need Help?

操作遇到問題時，先檢查這 3 件事

找不到 Pico

先確認 USB 線有插好，再請老師一起看解譯器有沒有選對。

按了 Run 沒反應

先確認是否已開啟 `keyboard-car-control.py`，再按一次綠色 `Run`。

打字沒反應

把游標點到下方 `Shell`，輸入 `w` 或 `x` 後再按 Enter。
