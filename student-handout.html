If You Get Stuck

卡住時先看這 4 件事

車子沒反應

先檢查 USB 是否插好，Pico 是否真的連到電腦。

Thonny 沒看到 Pico

確認右下角的 Interpreter 是否選到 MicroPython (Raspberry Pi Pico)。

按了 Run 但沒有動

確認你有先貼上完整程式，並且成功存檔執行。

小車一直動

先在 Shell 輸入 x，再按 Enter 讓小車停止。
