File Location

先分清楚：電腦本機端 vs Pico 板子端

學生只要記住一件事：下載檔會先在電腦裡，真正要被 Pico import 或開機執行的檔案，才需要用 Thonny 上傳到 Pico。

Pico

Raspberry Pi Pico 端

Raspberry Pi Pico /
├─ mango/                     ← 必須上傳，給 from mango import ... 使用
│  ├─ __init__.py
│  └─ ...
├─ main.py                    ← 只有要開機自動執行才存這個名字
└─ practice.py                ← 平常練習可用任意檔名
Class Flow

上課時學生照這樣做

  1. 先在電腦打開網站與下載區，確認今天要用的程式或 zip。
  2. 在 Thonny 左側檔案視窗分清楚 This computer 與 Raspberry Pi Pico。
  3. 把 mango 資料夾上傳到 Pico 根目錄；PDF、PPT、Markdown 不需要上傳到 Pico。
  4. 練習程式先貼到 Thonny 按 Run，確定成功後再決定是否存到 Pico。
  5. 只有希望 Pico 重新插電後自動跑，才把最後版本存成 Pico 上的 main.py。

Required Steps

開始前先完成 6 件事

這 6 步完成後，學生再看 01-05 才會像是在學小車，而不是一直處理環境錯誤。

Step 01接上 Pico 小車

用 USB 線連接電腦與 Raspberry Pi Pico，並確認小車放在平面上，輪子不要懸空亂轉。

Step 02打開 Thonny

右下角選擇 MicroPython (Raspberry Pi Pico)。如果看不到，先請老師協助安裝或更新 MicroPython。

Step 03上傳 mango 函式庫

下載 mango-library.zip，解壓縮後把整個 mango 資料夾放到 Pico 的根目錄。

Step 04執行 import 測試

貼上測試程式，看到 mango import OK 才進入 02-05 中需要 mango 的 cases。

Step 05跑馬達校正

先確認右輪、左輪、前進、後退、左右轉方向正確。若相反，修改 RIGHT_SIGN 或 LEFT_SIGN。

Step 06再進入正式主題

完成設定後，先做 BootCamp 或 01 入門；要真的控制小車移動時，再進入 03 小車移動。

Mango Library

mango 函式庫要先放進 Pico

許多主題會使用 from mango import Motor、RUS04、WS2812B、GraySensor8。如果沒有上傳函式庫，學生會看到 ImportError 或 ModuleNotFoundError。

Motor Calibration

馬達方向先校正，再談避障或循跡

如果左右輪方向錯了，後面的避障與循跡都會看起來像程式壞掉。先用這段程式確認每個方向，再進入主題 03。

馬達校正程式

先觀察每個動作是否符合中文提示；若方向相反，修改 RIGHT_SIGN 或 LEFT_SIGN。

from machine import Pin
import time

# Raspberry Pi Pico 小車馬達校正
# 教材統一腳位：
#   右馬達 M1 = GP12 / GP13
#   左馬達 M2 = GP11 / GP10

RIGHT_A = Pin(12, Pin.OUT)
RIGHT_B = Pin(13, Pin.OUT)
LEFT_A = Pin(11, Pin.OUT)
LEFT_B = Pin(10, Pin.OUT)

# 如果某一邊前進方向相反，把 1 改成 -1。
RIGHT_SIGN = 1
LEFT_SIGN = 1


def _set_motor(pin_a, pin_b, direction):
    if direction > 0:
        pin_a.value(1)
        pin_b.value(0)
    elif direction < 0:
        pin_a.value(0)
        pin_b.value(1)
    else:
        pin_a.value(0)
        pin_b.value(0)


def drive(left, right):
    _set_motor(LEFT_A, LEFT_B, left * LEFT_SIGN)
    _set_motor(RIGHT_A, RIGHT_B, right * RIGHT_SIGN)


def stop():
    drive(0, 0)


tests = [
    ("右輪往前", 0, 1),
    ("左輪往前", 1, 0),
    ("整台前進", 1, 1),
    ("整台後退", -1, -1),
    ("原地左轉", -1, 1),
    ("原地右轉", 1, -1),
]

print("Motor calibration start")
print("如果方向相反，先按 Ctrl+C 停止，再修改 RIGHT_SIGN 或 LEFT_SIGN。")

for label, left, right in tests:
    print(label)
    drive(left, right)
    time.sleep(1.2)
    stop()
    time.sleep(0.8)

print("Motor calibration done")
stop()
